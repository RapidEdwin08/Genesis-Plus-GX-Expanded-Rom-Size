Source Build Installation (RetroPie-Setup):
---------------------------------------------------
Place "lr-genesis-plus-gx-EX.sh" to the following directory on your Device
"/home/$USER/RetroPie-Setup/scriptmodules/libretrocores"

wget https://raw.githubusercontent.com/RapidEdwin08/Genesis-Plus-GX-Expanded-Rom-Size/master/builds/lr-genesis-plus-gx-EX.sh -P ~/RetroPie-Setup/scriptmodules/libretrocores/

1.Access setup script (Wifi or Ethernet connection required)
2.Select "Manage *Experimental* Packages"
3.Select "lr-genesis-plus-gx-EX"
4.Select "Update/Install from source"
5.Core will then compile and install
6.Results in new Core to Select from: lr-genesis-plus-gx-EX (for SegaChannel+P4PR1UM)
